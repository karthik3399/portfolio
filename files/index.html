01 — About

The short version.

A computer science foundation paired with a habit of asking "what does this data actually mean?"

I'm a final-year Computer Science Engineering student at Malla Reddy University, Hyderabad, with a steady interest in the place where software engineering meets data analysis. I like the part of a project where a pile of disconnected numbers turns into a chart someone can actually act on.

My toolkit spans Python, SQL, and Power BI for analytics work, and React, Node.js, and MongoDB for building the applications that data eventually lives in. I've also spent time with machine learning frameworks like Scikit-learn, TensorFlow, and PyTorch, exploring how predictive models fit into real-world products.

Outside of coursework, I build projects that force me to use these tools end-to-end — from a healthcare analytics dashboard to a blockchain-secured platform — because that's where the actual learning happens.
